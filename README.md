# Self-Evolution Skill System

**[中文文档](./self_evolution_cn.md)** | **[English](./self_evolution_en.md)**

> A framework that enables AI Agent's Skill system to automatically learn from execution experience and improve over time. Inspired by [OpenSpace](https://github.com/HKUDS/OpenSpace).

## Highlights

- **Automatic Trace Recording** — Capture reasoning, tool calls, and results
- **Intelligent Quality Analysis** — LLM-based evaluation of success rate and efficiency
- **Three Evolution Modes** — FIX (patch defects) / DERIVED (specialize) / CAPTURED (extract new skills)
- **Full Version Traceability** — DAG-based skill lineage for audit and rollback
